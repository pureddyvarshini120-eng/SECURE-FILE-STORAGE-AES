# Secure File Storage System with AES

## Overview
A Python-based application that securely encrypts and decrypts files using AES encryption. The project also verifies file integrity using hash verification.

## Features
- AES-256 file encryption
- Secure file decryption
- Metadata storage (file name, timestamp, hash)
- Integrity verification
- Command Line Interface (CLI)

## Technologies Used
- Python
- cryptography
- hashlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python main.py
